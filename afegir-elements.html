Afegir Elements

última actualització 01 de Gener de 2013

traducció 21 de Maig de 2015

Un dels teus primers serà fer servir D3 per crear un nou element del DOM. Normalment, serà un obejcte SVG per crear una visualització de dades, però començarem d'una manera més senzilla, i crearem un senzill paràgraf

Comencem amb aquesta senzilla plantilla HTML:

          
            <!DOCTYPE html>
            <html lang="en">
              <head>
                <meta charset="utf-8">
                <title>D3 Test</title>
               <script type="text/javascript" src="d3/d3.v3.js"></script>
              </head>
              <body>
                <script type="text/javascript">
                  // Aquí escriuràs el teu meravellòs codi javascript
                </script>
              </body>
            </html>
